# Automated Billing System

A professional Python billing software for supermarkets, shops and retail stores.

## Features

- Inventory Management
- Customer Details
- Billing
- GST
- Discount
- Invoice Generator
- Bill History
- Sales Reports

## Installation

```bash
git clone https://github.com/yourusername/automated-billing-system.git

cd automated-billing-system

pip install -r requirements.txt

python src/main.py
```
